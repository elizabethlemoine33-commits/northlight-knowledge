# Northlight Knowledge

> A living research notebook for Northlight Advisory Services.

## Purpose

This repository is the canonical source for the ideas, observations, evidence, and frameworks that emerge through practice.

It is not a publishing platform.

It is not a website.

It is the place where thinking develops before it becomes articles, client work, presentations, or software.

## Philosophy

Northlight approaches organisations as living systems rather than machines.

This repository reflects that philosophy.

Ideas begin as observations.

Repeated observations become current understandings.

Current understandings supported by consistent evidence may eventually become core principles.

Nothing here is considered immutable.

Everything is open to refinement as new evidence emerges.

## Knowledge Flow

```
Practice
    ↓
Observations
    ↓
Evidence
    ↓
Current Understandings
    ↓
Core Principles
    ↓
Frameworks
    ↓
Published Work
```

Published work feeds new conversations, new evidence, and new observations back into the repository.

## Repository Structure

- `/worldview` — foundational beliefs and values
- `/observations` — field notes from practice
- `/evidence` — supporting material from blogs, conversations, case studies, and research
- `/principles` — current understandings and mature principles
- `/frameworks` — practical models derived from experience
- `/corpus-reviews` — analyses of published work over time
- `/open-questions` — ideas still under investigation

## Working Principles

- Observe before concluding.
- Separate evidence from interpretation.
- Record uncertainty.
- Revise when better evidence appears.
- Build from practice rather than theory alone.

## Status

This repository is intended to evolve continuously as Northlight grows.
