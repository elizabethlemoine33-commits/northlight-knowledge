# Organisational Clarity Framework

## Purpose
Improve execution by increasing organisational clarity.

## Sequence
1. Clarify outcomes.
2. Clarify ownership.
3. Clarify decisions.
4. Clarify information flow.
5. Clarify measures.

## Expected Outcomes
- Faster decisions
- Less rework
- Better alignment
- Increased capacity
